Math.round(x) x 값에서 제일 근접한 정수를 반환 Math.pow(x,y) x의 값을 y의 횟수만큼 제공한 값을 반환 Math.sqrt(x) x의 제곱근 값을 반환 Math.abs(x) x의 양수값을 반환 Math.ceil(x) x의 값에서 소수점 아래 값을 무조건 올림해서 정수반환 Math.floor(x) x의 값에서 소수점 아래 값을 무조건 버린 정수 반환 Math.min() 인수로 넣은 값들 중에서 제일 작은 값을 반환 Math.max() 인수로 넣은 값들 중에서 제일 큰 값을 반환 Math.random() 0에서 1사이의 실수를 랜덤으로 반환 Math.floor(Math.random() * num) 특정수까지의 랜덤한 정수를 반환
